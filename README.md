# SOC Alert Investigation Playbooks

This repository contains structured investigation playbooks for common Security Operations Center (SOC) alerts.

The goal of this project is to demonstrate practical incident investigation workflows used by SOC analysts when responding to security alerts.

Playbooks included in this repository cover:

- Brute Force Login Attacks
- Phishing Email Investigation
- Suspicious PowerShell Activity
- Impossible Travel Logins
- Data Exfiltration Alerts

Each playbook outlines the triage process, investigation steps, containment actions, and detection improvement strategies.

These playbooks are designed to simulate real-world SOC incident response procedures.
